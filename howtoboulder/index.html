    Проведение болулдеринга

    Проведение соревнований в дисциплине боулдеринг

    • Создать соревнование
      • Создать группы
      • Привязать главного судью и главного секретаря.
    • Переодически проверять список регистрирующихся спортсменов
      • После проверки/исправления данных нажимаем кнопку "Опубл."

    Важно

    Опубликованный спортсмен отображается в списке заявившихся, который виден всем. Строка опубликованной заявки меняет цвет на синий

    • После завершения приема заявок

      • Подготовить списки участников по сетам для судьи на регистрации
      • Распечатать и порезать номера участников. (Печатаем 4 на лист).
      • Распечатать и порезать карточки участника. (Печатаем 4 на лист).
      • Распечатать журнал инструктажа по ТБ.

    • Судья на регистрации отмечает уастников, выдает номера, карточки участника, заполняет журнал ТБ (подписи), по окончании отдает списки явившихся секретарю

    • Секретарь в квалификации

      • Готовит стартовые протоколы и протоколы трасс для судей на трассах
      • по спискам, в разделе регистрация добавляет участников в соревнования
      • По окончанию раунда вносит результаты (результаты вносим с карточек, по необходимости сверяемся с протоколом)
      • Проверяет, распечатывает и подписывает протоколы
      • Вывешивает протоколы на стенд информации

    Внимание

    Обычно соревнования проходят в фестивальном формате по сетам. Стартовые протоколы готовим по сетам. Протоколы пока готовим вручную, сортируем по полу, потом по алфавиту. Можно подготовить протоколы заранее по предварительной регистрации, потом удалить тех кто не пришел.

    Внимание

    Участников прошедших в финал отмечать вручную в Excel файле протокола. Например: очищаем графу "Вып. Раз." и проставляем туда значение "Q".

    • Секретарь в финальном раунде

      • Отмечает в разделе "Результаты Q" участников вышедших в финал
      • Готовит стартовый протокол и протокол трассы на финальный раунд
      • Вывешивает стартовый протокол в зоне изоляции
      • По окончанию раунда вносит результаты
      • Проверяет, распечатывает и подписывает протоколы
      • Вывешивает протоколы на стенд информации

    • Секретарь по завершении всех раундов соревнований

      • Подготавливает и проверяет протоколы
      • Подписывает протоколы сам и отдает на подпись главному судье
      • Ставит на протоколы печать проводящей организации
      • Подготавливает документ - "Состав судейской коллегии". Пункт меню "Судьи/Судейская бригада"
      • По необходимости подготавливает справки о судействе. Пункт меню "Судьи/Судейская бригада"
      • Публикует протоколы в электронном виде

    Внимание

    Необходимо проверить расчёт разрядов и соответствие их присвоения уровню соревнований
